# Projekt
Závěrečný projekt- automatizované hlídání podmínek v terráriu

1. Nakoupení komponentů
2. Práce s knihovnami v Arduinu
3. Zapojeni na nepájivém poli
4. Práce s MQTT serverem a HomeAssistent
